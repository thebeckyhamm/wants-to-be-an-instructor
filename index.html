Philosophy

Design isn't only about looking pretty. Looks are just part of the total experience when a user visits a website or application. Ease-of-use (UX), performance, and accessibility are all necessary when building a website today.

A Multi-Device World

These days mobile first is the expectation. If a user wants to complete a task on his phone, he has every right to expect that it will work. Saying "No one would ever try X on their phone" is not only incorrect, but also short-sighted.
